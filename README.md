lirc
====

[LIRC](http://www.lirc.org/) configuration files to make Raspberry Pi work as a remote

## Environment

- Raspberry Pi 3 Model B
- Raspbian Jessie
- `lirc` (`0.9.4c-9`)
- `liblirc0` (`0.9.4c-9`)
- `liblirc-client0` (`0.9.4c-9`)

## Installation

Put the configuration files in `/etc/lirc/`.
